# CodeAlpha Bioinformatics Internship

This repository contains the bioinformatics tasks completed as part of the CodeAlpha Internship.

## Completed Tasks

### Task 1: DNA/Protein Sequence Analysis
- Retrieved protein sequence from UniProt.
- Performed BLAST analysis.
- Documented sequence similarity and alignment results.
- Included screenshots and analysis report.

### Task 2: Multiple Sequence Alignment
- Retrieved insulin protein sequences from Human, Pig, Rat, Mouse, and Chicken.
- Performed Multiple Sequence Alignment using Clustal Omega.
- Identified conserved regions and sequence similarities.
- Included alignment screenshots and report.

### Task 3: Protein Structure Prediction
- Selected Human Insulin (P01308) from UniProt.
- Visualized the 3D protein structure using SWISS-MODEL.
- Analyzed the protein structure from different views.
- Included structure images and report.

## Tools Used
- UniProt
- NCBI BLAST
- Clustal Omega
- SWISS-MODEL
